# Quote Readiness

Read `../../core/source-routing.md`, `../../core/safety-gates.md`, and
`../../core/output-schemas/quote_readiness.md`.

Steps:
1. Summarize customer requirements and constraints.
2. Decide whether the material is ready for quotation work.
3. List missing inputs, risk notes, and handoff items.
4. Keep quotation generation in the quotation workflow; SalesFlow only prepares the handoff.
