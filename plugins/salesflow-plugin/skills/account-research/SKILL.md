# Account Research

Read `../../core/source-routing.md`, `../../core/safety-gates.md`, and
`../../core/output-schemas/customer_research.md`.

Steps:
1. Identify the customer using hard identifiers where possible.
2. Gather sales memory, source evidence, and optional public evidence.
3. Label key claims with evidence levels.
4. Separate confirmed facts, likely judgments, unknowns, and next actions.
5. Produce customer research output and a `write_plan` only if persistence is requested.
