# Competitive Brief

Read `../../core/source-routing.md`, `../../core/safety-gates.md`, and
`../../core/output-schemas/competitive_brief.md`.

Steps:
1. Identify the competitor, claim, or objection.
2. Prefer reusable sales knowledge and official evidence over hearsay.
3. Separate customer-specific negotiation facts from reusable talk tracks.
4. Produce a sales knowledge write plan only for reusable, verified content.
