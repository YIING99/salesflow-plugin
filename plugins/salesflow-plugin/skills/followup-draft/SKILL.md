# Follow-Up Draft

Read `../../core/source-routing.md`, `../../core/safety-gates.md`, and
`../../core/output-schemas/followup_draft.md`.

Steps:
1. Confirm the customer and last-touch evidence.
2. Draft plain-text customer-facing copy without a signature block by default.
3. Keep internal judgment separate from customer-facing text.
4. If the user later confirms the email was sent, produce a customer activity write plan.
