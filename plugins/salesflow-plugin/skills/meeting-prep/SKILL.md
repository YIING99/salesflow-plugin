# Meeting Prep

Read `../../core/source-routing.md`, `../../core/safety-gates.md`, and
`../../core/output-schemas/meeting_prep.md`.

Steps:
1. Confirm meeting goal, customer identity, and available history.
2. Summarize context, risks, likely objections, and desired next step.
3. Prepare questions and talk tracks with evidence levels for factual claims.
4. Do not write by default; create a write plan only when the user asks to persist outcomes.
