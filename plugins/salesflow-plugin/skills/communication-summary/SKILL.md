# Communication Summary

Read `../../core/source-routing.md`, `../../core/safety-gates.md`, and
`../../core/output-schemas/communication_summary.md`.

Steps:
1. Confirm source channel, time range, and customer identity evidence.
2. Extract facts, judgments, risks, commitments, and next actions.
3. Split persistence candidates into activity, stable profile, reusable sales knowledge, and personal learning.
4. Produce a write plan for any durable update; wait for confirmation before writing.
